Here is where you would extol the benefits of membership along with a special link to the checkout page. If you had multiple subscription plans, you could have a pricing table with the different plans, features, and prices.

To take the user to the checkout page for a particular plan, you need to create a link to the path where the paywall plugin is mounted in your aerobatic.yml. Passing the Chargebee plan_id and mode=init_checkout as query parameters.
